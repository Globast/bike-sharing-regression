# 12. Referencias

Las referencias se gestionan mediante un archivo BibTeX (`references.bib`). A continuación se muestran algunos ejemplos de citas dentro del texto:

- Análisis clásico de series temporales [@boxjenkins2016].
- Introducción práctica a las series temporales con R [@hyndman2018].
- Descripción original del conjunto de datos de *Bike Sharing* [@fanaee2014].
- Discusión general sobre regresión y gráficos [@cleveland1990].

En el resto de capítulos, las citas se incluyen con el formato `[ @clave ]` (sin espacios), por ejemplo `[@fanaee2014]`.
